# aliyunvodupload
因为在https://packagist.org/找不到相应的包，因此自己将阿里云的VOD UPLOAD SDK 打包上传 方便使用
下载方法 composer require jifangshiyanshi/aliyunvodupload @dev

这个包是从服务器上传视频 因此用的少，后来发现ali已经有composer包了
composer require alibabacloud/sdk
